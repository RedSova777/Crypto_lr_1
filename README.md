# Farey Left Neighbor

Решение задачи поиска числителя дроби, непосредственно стоящей слева от
`3/7` среди несократимых дробей со знаменателем не более `1000000`.
  
  
## Запуск

```bash
python3 farey_left_neighbor.py
```
  
Ожидаемый результат:
  
```text
Левый сосед: 428570/999997
Искомый числитель: 428570
```
